This is a streamlit based web application that recommends Netflix shows/movies based on user preferences(age_criteria,genre,imdb_rating and type) or title. 

Dataset: Kaggle Netflix Dataset. 


Features:
- Cleaned and pre-preprocessed dataset
- Uses One-hot encode and Multi-hot encode
- Uses KNN and Cosine Similarity for recommendations 
- Streamlit based interactive web application

TechStack:
- Python
- Pandas, Scikit-learn
- Streamlit
- Cosine Similarity, KNN
- MultiLabelBinarizer, MinMaxScaler
